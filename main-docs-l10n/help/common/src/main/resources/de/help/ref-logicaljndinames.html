Logische JNDI-Namen

Bei der Erstellung einer Ressource geben Sie einen eindeutigen JNDI-Namen für die Ressource an. Mithilfe dieses Namens greifen Anwendungen auf die Ressource zu.

Der Java EE-Standard legt fest, dass Anwendungen bestimmte Standardressourcen verfügbar gemacht werden müssen, und definiert bestimmte JNDI-Namen für diese Standardressourcen. GlassFish Server stellt diese Namen über logische JNDI-Namen bereit, welche Java EE-Standard-JNDI-Namen bestimmten GlassFish Server-Ressourcen wie folgt zuordnen:

java:comp/DefaultContextService

Dieser Java EE-Standardname wird der Ressource concurrent/__defaultContextService zugeordnet.

java:comp/DefaultManagedThreadFactory

Dieser Java EE-Standardname wird der Ressource concurrent/__defaultManagedThreadFactory zugeordnet.

java:comp/DefaultManagedExecutorService

Dieser Java EE-Standardname wird der Ressource concurrent/__defaultManagedExecutorService zugeordnet.

java:comp/DefaultManagedScheduledExecutorService

Dieser Java EE-Standardname wird der Ressource concurrent/__defaultManagedScheduledExecutorService zugeordnet.

java:comp/DefaultDataSource

Dieser Java EE-Standardname wird der Ressource jdbc/__default zugeordnet.

java:comp/DefaultJMSConnectionFactory

Dieser Java EE-Standardname wird der Ressource jms/__defaultConnectionFactory zugeordnet.
