JDBC

使用「JDBC」頁面可組態下列類型的資源：

  • JDBC 資源

  • 連線集區

Java Database Connectivity (JDBC) 技術提供關聯式資料庫的連結。在 GlassFish Server 中，可以透過將 JDBC 資源 (資料來源) 增加至網域來組態資料庫連結。資料來源是 Java Platform, Enterprise Edition (Java EE 平台) 組態資料庫連結的標準方法。每個資料來源都包含一個資料庫連線集區。應用程式會在 Java Naming and Directory Interface (JNDI) API 樹狀結構或本機應用程式環境中查詢資料來源，然後使用 getConnection 方法擷取資料庫連線。資料來源及其連線集區提供了有助於保持系統執行的連線管理處理作業。

GlassFish Server 包括一個 Java DB 資料庫，且已預先組態了存取此資料庫的功能。
