JDBC

Utilizzare la pagina JDBC per configurare risorse dei tipi seguenti:

  • Risorse JDBC

  • Connection pool

La tecnologia JDBC (Java DataBase Connectivity) fornisce le funzioni di connettività ai database relazionali. In GlassFish Server è possibile configurare la connettività ai database mediante l'aggiunta di risorse JDBC (origini dati) al dominio. Per origine dati si intende un metodo standard utilizzato dalla piattaforma Java EE (Java Platform Enterprise Edition) per la configurazione della connettività a un database. Ogni origine dati contiene un pool di connessioni al database. Le applicazioni cercano l'origine dati nella struttura dell'interfaccia API JNDI (Java Naming and Directory Interface) o nel contesto applicazione locale, quindi recuperano una connessione al database con il metodo getConnection. Le origini dati e i relativi connection pool forniscono processi di gestione delle connessioni che garantiscono il funzionamento del sistema.

GlassFish Server include una copia del database Java DB ed è preconfigurato con la funzione di accesso a tale database.
